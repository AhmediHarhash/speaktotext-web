# SpeakToText Web

Standalone landing page repo for the public website.

## Local preview

Open `index.html` directly or serve the folder with any static server.

Example:

```powershell
cd "C:\Users\PC\Desktop\website website hustle\Projects\speaktotext-web"
npx serve .
```

## Deploy

Recommended hosting: Vercel

- Import this repo into Vercel
- Add the custom domain `speaktotext.hekax.com`
- In Cloudflare, add the DNS record Vercel gives you
